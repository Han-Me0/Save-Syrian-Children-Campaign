# Save Syrian Children Campaign Website

This is a responsive HTML & CSS campaign website created to raise awareness about the educational crisis faced by children in Syria who have dropped out of school due to the ongoing war.

## Table of Contents
- [Overview](#overview)
- [Features](#features)
- [Technologies Used](#technologies-used)
- [Screenshots](#screenshots)
- [How to Use](#how-to-use)


## Overview

The "Save Syrian Children" website is a fully responsive site designed to spread awareness and garner support for children affected by the Syrian war who have lost access to education. The site is built using only HTML and custom-made CSS, without any frameworks.

## Features

- Responsive Design: Works on all device sizes (desktop, tablet, mobile).
- Eye-Catching Layout: Engages users with powerful visuals and messaging.
- Multimedia Elements: Includes impactful images and animations.
- External Sources: Uses Google Fonts for typography and FontAwesome for icons.
  
## Technologies Used

- HTML5: Markup for structuring the website.
- CSS3: Custom-made styles for the layout and design.
- External Fonts: Google Fonts for enhanced typography.
- Font Awesome: For social icons and visual elements.
